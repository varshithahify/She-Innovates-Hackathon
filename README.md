# She-Innovates-Hackathon
Rural Material Mortality Due to Lack of Continuous Monitoring
Addresses lack of continuous maternal monitoring in rural areas

Uses AI to predict complications before emergencies occur

Combines embedded systems, IoT, and real-time analytics

Enables proactive rather than reactive healthcare

Reduces maternal and infant mortality risk

Affordable and scalable for rural deployment

Supports healthcare workers with data-driven insights

Strengthens overall rural healthcare infrastructure
